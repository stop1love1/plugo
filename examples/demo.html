Plugo Demo

Đây là website mẫu với Plugo chat widget. Nhấn vào bubble chat ở góc phải để thử!

Về Plugo

Plugo là một embeddable AI chat widget cho phép bất kỳ website nào thêm trợ lý AI thông minh chỉ bằng 3 dòng code. Bot tự học nội dung website, hiểu context trang hiện tại và thực hiện action thực tế.

Tự động crawl

Bot tự học toàn bộ nội dung website, không cần config thủ công.

Multi-LLM

Hỗ trợ Claude, GPT-4o, Gemini, Ollama. Tuỳ chọn theo nhu cầu.

API Tools

Bot gọi API website để thực hiện action: tìm kiếm, đặt hàng, đăng ký...

Hướng dẫn sử dụng

1. Tạo tài khoản tại Dashboard

2. Nhập URL website để crawl nội dung

3. Copy script tag và dán vào website

4. Widget chat tự động xuất hiện!
